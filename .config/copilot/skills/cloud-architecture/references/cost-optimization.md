# Cost Optimization

- Right-size compute, storage, and database resources.
- Plan reserved capacity and savings plans.
- Use spot/preemptible capacity for fault-tolerant workloads.
- Implement autoscaling and schedule-based scaling.
- Apply storage lifecycle policies and archival tiers.
- Optimize network egress and inter-region traffic.
- Rationalize licensing and marketplace usage.
- Establish FinOps practices, budgets, and anomaly detection.
